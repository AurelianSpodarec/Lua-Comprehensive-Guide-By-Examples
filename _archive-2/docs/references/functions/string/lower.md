---

    title: string.lower() 

---

The `string.lower` function converts all uppercase letters in a string to lowercase.

### Syntax  
```lua
string.lower(s)
```  

### Parameters  

#### `s`  
The string to convert.  

### Examples  

#### Basic usage.
```lua
local result = string.upper("HELLO WORLD!")
print(result)

-- Output: hello world!
```
