---

    title: string.reverse() 

---

The `string.reverse` function returns a string with its characters in reverse order.

### Syntax  
```Lua
string.reverse(s)
```  

### Parameters  

#### `s`  
The string to reverse.  

### Examples  

#### Basic usage.
```Lua
local result = string.reverse("Lua")
print(result)

-- Output: auL
```
